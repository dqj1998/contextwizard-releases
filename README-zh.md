## 🌟 产品概述

**ContextWizard** 是一款免费、本地优先的浏览器扩展，可自动捕获、整理、搜索和复用你在 **ChatGPT、Claude、Gemini、Copilot、Perplexity、Grok、DeepSeek 等 10+ AI 平台** 上的对话——创建一个统一、可搜索的工作空间，随你的项目一同成长。

> **不再丢失 AI 洞见。ContextWizard 打破平台间的信息孤岛——你在 Claude 上的对话上下文，可以随你一起带到 ChatGPT。**

| ✅ 自动捕获 | ✅ 跨平台搜索 | ✅ 书签与提醒 |
|:---:|:---:|:---:|
| ✅ 上下文面板 v2.0 | ✅ 工作空间 | ✅ 100% 本地与隐私 |

## ✨ 核心功能

### 📸 自动捕获
对话在你聊天时自动保存。无需按钮，无需手动操作。扩展程序实时检测所有支持平台上的新消息，并内置智能去重。

- **自动模式**：持续保存每次对话
- **域名过滤**：选择要监控的 AI 平台
- **平台适配器**：为每个 AI 服务提供精确的内容提取
- **视觉指示器**：随时了解捕获状态

### 🔍 强大的跨平台搜索
**一键搜索所有平台的所有已捕获对话**。

- 短语搜索 + 关键词匹配
- 按相关性和时间智能排序
- 按平台、工作空间或时间段筛选
- 上下文预览，无需打开即可查看
- 匹配类型指示器，清晰显示匹配原因

> 一次搜索即可同时查询 ChatGPT、Claude、Gemini 和所有其他平台。不再需要逐个平台查找。

### 🔖 跨平台书签与分组
在统一界面中保存跨 ChatGPT、Claude 和 Gemini 的关键对话。将书签整理到自定义分组中。

- **嵌套书签组**：标记到具体消息，而非仅对话链接
- **自动生成标题**：分组标题从对话内容自动生成
- **智能清理**：空分组自动移除

### 🧠 上下文面板（v2.0）
**书签上下文聊天**——你保存的洞见不再留在原地。开启新对话时，可直接注入相关书签。

- **浮动面板**出现在聊天输入框旁
- 选择书签直接注入到当前对话
- **智能推荐**：自动显示与当前话题最相关的书签
- **会话管理**：每个项目保持独立的上下文
- 键盘导航（Tab、方向键、空格、回车、Esc）

### 📂 AI 工作空间
将对话按项目整理到不同工作空间中。

- 拖拽式管理工作空间
- 新捕获自动归入最近使用的工作空间
- 工作空间级搜索，聚焦单个项目
- 分组标题从对话内容自动生成

### ⏰ 智能提醒
再也不会丢失重要洞见。在任何已书签的消息上设置提醒。

- **重复规则**：一次、每天、工作日、每周、每月
- **浏览器通知**，即使窗口最小化也能可靠触发
- 点击通知直接跳转到书签消息

### ☁️ 可选云同步（付费）
在多台电脑之间同步工作空间和书签。

- 传输中和静态数据均端到端加密
- Google SSO 登录——无需第三方服务
- 7 天免费试用，无需信用卡

### ✏️ 提示词编辑器与隐私保护
管理可复用的提示词模板，内置自动脱敏功能。

- 创建、编辑和组织提示词
- **自动脱敏**敏感信息（API 密钥、密码、令牌）
- 自定义脱敏规则

## 🚀 快速开始

1. **安装扩展**：从 [Chrome 应用商店](https://chromewebstore.google.com/detail/contextwizard-%E2%80%93-ai-worksp/lmhnmmedgmnfggecdalkancllnekofnb) 或 [Edge 插件商店](https://microsoftedge.microsoft.com/addons/detail/contextwizard/nknoacgaapoeboehlgelolgbifgcimli) 安装
2. **打开面板**：点击工具栏中的 ContextWizard 图标
3. **选择平台**：开启你想要捕获的 AI 平台（默认全部开启）
4. **开始聊天**：访问任一支持的 AI 平台——捕获自动开始
5. **搜索与整理**：搜索、书签、组织你的对话

> 无需账号。无需 API 密钥。无需配置文件。安装即用。

## 🛡️ 隐私与安全

**ContextWizard 采用本地优先、零信任架构构建。**

| 特性 | 详情 |
|------|------|
| **数据存储** | 所有数据本地存储在 IndexedDB（`chrome.storage.local`） |
| **云端上传** | 默认无。仅在你明确启用时进行可选云同步 |
| **加密** | 使用云同步时端到端加密 |
| **跟踪** | 零跟踪。无分析，无遥测 |
| **权限** | Manifest V3 最小权限——仅限 AI 平台域名 |
| **账户** | 核心功能无需账户 |

你的提示词、专有代码、业务上下文和个人信息**默认完全离线**。

## 🏗️ 技术亮点

- **Manifest V3** Chrome 扩展，TypeScript 代码库，esbuild 打包
- **WebAssembly 加速** 全文检索引擎，数千条对话毫秒级响应
- **平台特定内容适配器**，基于 MutationObserver 和 CSS 选择器
- **通用 MessageExtractor** 后备机制，支持自定义平台
- **智能去重**，防止重复保存已查看的对话
- **自动截图**，为每条保存的对话生成缩略图
- **防抖内容保存**，平衡响应速度与存储效率
