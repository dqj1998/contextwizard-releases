# ContextWizard v2.0.0 — Bookmark Context Chat

🎉 **Released: July 2, 2026**

## What's New in 2.0

### 🧠 Context Panel (Major New Feature)
Your bookmarked insights no longer stay behind when you start a new conversation. The Context Panel appears next to your chat input, letting you select and inject relevant bookmarks directly into any conversation.

- **Floating Panel**: Access bookmarks without leaving the page
- **Smart Relevance**: Automatically shows the most relevant bookmarks for your current topic
- **Session Management**: Each project/tab keeps its own context
- **Keyboard Navigation**: Tab, arrows, Space, Enter, Escape

### 🔧 Popup Redesign
- New tab-based layout for faster navigation
- Conversations and bookmarks one click away

### 📁 Bookmark Group Enhancements
- Rename groups directly
- Start a chat right from bookmark list
- One-click jump back to bookmarked conversations

### 🛡️ No New Permissions Required
v2.0.0 adds no new permissions. All features remain local by default.
