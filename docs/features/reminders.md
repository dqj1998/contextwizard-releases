# Smart Reminders

Never lose a valuable insight. Set reminders on any bookmarked message to revisit it later.

## Recurrence Options

| Option | Behavior |
|--------|----------|
| **Once** | Single notification at the scheduled time |
| **Daily** | Every day at the set time |
| **Weekdays** | Monday through Friday at the set time |
| **Weekly** | Every week on the selected day |
| **Monthly (by date)** | Every month on the selected date |
| **Monthly (by weekday)** | Every month on the selected weekday (e.g., "second Tuesday") |

## Notifications

- **Browser notifications** fire reliably at the scheduled time
- Works even when the browser is minimized
- **Click the notification** to jump directly to the exact bookmarked message
- Notifications respect your system's Do Not Disturb settings

## Setting a Reminder

1. Open a conversation that has bookmarked messages
2. Click the **bell icon** (🔔) on any bookmarked message
3. Choose your recurrence pattern
4. Set the time and date
5. Click **Save**

## Managing Reminders

- View all active reminders in the dashboard
- Edit or delete reminders at any time
- Notification history is available for review
