# Context Panel (v2.0)

**Bookmark Context Chat** — Your bookmarked insights no longer stay behind when you start a new conversation. Available in ContextWizard v2.0.0+.

## Overview

The Context Panel bridges the gap between saving insights and using them. Instead of manually copying and pasting bookmarked content into new conversations, the panel lets you select bookmarks and inject them directly into your current AI chat — without leaving the page.

## How It Works

1. Open any supported AI platform (ChatGPT, Claude, Gemini, etc.)
2. Click the ContextWizard icon or use the keyboard shortcut
3. The floating panel appears next to the chat input
4. Browse or search your bookmarks
5. Select the ones relevant to your current topic
6. Click **Inject** — bookmark content is inserted into the chat

## Smart Relevance

The panel automatically shows the bookmarks most relevant to what you're currently discussing. Content is trimmed to fit within conversation context limits.

## Session Management

Working on multiple projects? Each browser tab and bookmark group keeps its own session:

| Tab | Bookmark Group | Context |
|-----|---------------|---------|
| Tab 1: Project A | Research notes | Independent of Tab 2 |
| Tab 2: Project B | Code snippets | Independent of Tab 1 |

## Keyboard Navigation

| Key | Action |
|-----|--------|
| Tab | Move between bookmarks |
| Arrow keys | Navigate list |
| Space | Toggle select bookmark |
| Enter | Inject selected bookmarks |
| Escape | Close panel |
