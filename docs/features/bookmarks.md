# Bookmarks & Groups

Save key moments across ChatGPT, Claude, Gemini, and other platforms in one unified place.

## Message-Level Bookmarks

Unlike browser bookmarks (which save only URLs), ContextWizard bookmarks let you mark the **exact message** within a conversation that matters.

- Bookmark any individual message in any captured conversation
- Each bookmark preserves the message content and context
- Jump back to the exact bookmarked message with one click

## Bookmark Groups

Organize bookmarks into named groups for easy retrieval.

- Create groups for different topics or projects
- Drag-and-drop bookmarks between groups
- **Auto-generated titles**: Group titles can be generated from bookmark content
- **Smart cleanup**: Empty groups are auto-removed
- **Rename groups** directly from the UI

## Cross-Platform Bookmark Management

All bookmarks are stored in a single interface, regardless of which AI platform the original conversation was on.

| Action | How |
|--------|-----|
| **Bookmark a message** | Open conversation → click bookmark icon → select message → choose or create group |
| **View all bookmarks** | Open dashboard → Bookmarks tab |
| **Jump to source** | Click any bookmark → opens original conversation at exact message |
| **Delete a bookmark** | Hover → delete icon (undo supported) |
| **Rename a group** | Click group name → edit |
| **Start a chat from a bookmark** | Bookmark list → "Start Chat" button |
| **Open in Bookmark Manager** | Hover preview or chat list → "Open in Bookmark Manager" button (v2.1) |
