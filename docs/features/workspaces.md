# AI Workspaces

Organize conversations into project-specific workspaces. Each workspace brings together ChatGPT threads, Claude chats, bookmarks, and search across everything you've discussed on that subject.

## Features

| Feature | Description |
|---------|-------------|
| **Project grouping** | Group conversations by topic, not by platform |
| **Drag-and-drop** | Move conversations between workspaces easily |
| **Auto-select** | Recent workspace auto-selects for new captures |
| **Auto-title** | Group titles generate from conversation content |
| **Auto-cleanup** | Empty groups auto-remove |
| **Workspace search** | Scope search results to a single workspace |

## How to Use Workspaces

### Create a New Workspace
1. Open the ContextWizard dashboard
2. Click **"New Workspace"**
3. Enter a name (e.g., "Marketing Campaign", "API Design", "Research Paper")
4. Start capturing conversations — they'll be tagged to this workspace

### Organize Conversations
- **Drag and drop** conversations between workspaces
- Conversations auto-tag to the most recently used workspace
- Each conversation can belong to one workspace

### Search Within a Workspace
- Switch to workspace-level search to scope results
- Only conversations in that workspace are searched
- Keeps project research focused and manageable
